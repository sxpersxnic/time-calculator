Right-click each icon below and "Save Image As" to download as PNG. Make sure to name them icon-192.png and icon-512.png respectively.

192x192 Icon

512x512 Icon

Instructions:

  1. Right-click on each SVG icon above
  2. Select "Save Image As..." or "Download Image"
  3. Save them as icon-192.png and icon-512.png
  4. Place them in the root directory of your project

Note: If your browser doesn't allow direct PNG export, you can:

  • Use an online SVG to PNG converter
  • Or use a tool like Inkscape or Adobe Illustrator to export at the correct sizes
  • Or use the developer tools to screenshot the SVGs
